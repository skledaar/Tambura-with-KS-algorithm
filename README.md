BACHELOR THESIS
TAMBURA SOUND SYNTHESIS USING THE KARPLUS-STRONG ALGORITHM

This paper examines the application of the Karplus-Strong algorithm to synthesize sounds of the tambura. It discusses how the algorithm output varies depending on the input excitement. Using the JUCE framework, a refined version of Karplus-Strong was implemented, with added support for parameter control to dictate the behavior of string synthesizers. This solution bypasses the issue of buffer size being an integer and it affecting the frequency of the generated note. An added element of realism is achieved by using recorded excitation samples from real instruments and enabling control of variation in picking speed and strength.

Mentor: prof. Ph.D. Kristian Jambrošić

To start the application, you need to place the .exe in the same folder as the Samples folder.
