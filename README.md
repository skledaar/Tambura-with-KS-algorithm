ZAVRŠNI RAD

Razvoj VST dodatka za sintezu zvuka tambure

Mentor: prof. dr. sc. Kristian Jambrošić

Rad se bavi primjenom Karplus-Strong algoritma na sintezu zvukova tambure. Raspravlja o problematici cjelobrojne veličine međuspremnika i kako to utječe na preciznost generiranja tonova. Pomoću razvojne okoline JUCE, implementira se dorađena verzija Karplus-Strong algoritma i dodaje kontrola parametara koji diktiraju ponašanje sintesajzera. Dodatna realnost zvuka postiže se korištenjem snimljenih pobuda s pravih instrumenata te parametrima za varijaciju brzine i jačine trzanja.
